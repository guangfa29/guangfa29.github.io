一、vue2.0

  1. Windows安装node.js，官网下载

  2. 获取nodejs模块安装目录访问权限

    sudo chmod -R 777 /usr/local/lib/node_modules/

  3. 安装镜像

    npm官方原始镜像网址是： https://registry.npmjs.org/

    npm config set registry https://registry.npmjs.org/

  4. 使用 cnpm 命令来安装模块

    cnpm install [name]

  5. 安装webpack

    cnpm install webpack -g

  6. 安装vue脚手架

    npm install vue-cli -g

  7. 找一个文件夹放工程，在终端中进入该目录

    cd 目录路径

  8. 根据模板创建目录

  9. 会有一些初始化的设置

  10. cd命令进入创建的工程

  11. 安装项目依赖，建议从官方仓库安装，因为npm服务器在国外这一步会比较慢

  12. 启动项目

    npm run dev
